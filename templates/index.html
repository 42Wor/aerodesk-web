Release Version Control Logs

v1.1.0 Stable June 2026

Introduced the new interactive console configuration wizard (`aerodesk config`) and dedicated screen detection layout manager (`aerodesk monitors`). Added automatic, parallel thread generation to spawn independent rendering daemons across multiple displays to support multi-monitor setups.

v1.0.0 Stable June 2026

Migration from the legacy live-wallpaper system to the unified AeroDesk platform. Added full Go-based CLI modules supporting cross-platform (Linux/macOS/Windows) execution, automated 90-day cached retention, and interactive portfolio linkages.

Pre-Release Build May 2026

Initial experiments with Shell automation pipelines and local unzip automation setups before compiling binary components.
